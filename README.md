# Booking Event
Event를 생성하고 Event 참석을 예약 및 취소하는 애플리케이션입니다.

## 기능
- 로그인
- 회원가입
- 이벤트 조회
- 이벤트 생성
- 이벤트 예약
- 이벤트 예약 취소

## 사용 기술
- Front-End : React
- Back-End : Node Express, Graphql
- Database : MongoDB
- Deployed : Heroku

## Installation and start
1. 레포지토리의 소스코드를 내려 받는다.
2. npm i
3. npm start
