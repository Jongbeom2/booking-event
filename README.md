# Booking Event
Event를 생성하고 Event 참석을 예약하는 애플리케이션입니다.

## 
Front-End : React
Back-End : Node Express, Graphql
Database : MongoDB
Deployed : Heroku

## Installation and start
1. 레포지토리의 소스코드를 내려 받는다.
2. npm i
3. npm start
